# ZanellaSpeed ⚡ v2.0

Dashboard profissional de análise de performance web.

## Funcionalidades

- 🏠 Home com hero, campo URL e histórico de análises
- 📄 Página de relatório dedicada por análise (`/report/[id]`)
- 📱 Análise simultânea Mobile + Desktop
- 🚀 Diagnóstico Rápido — agrupado por tipo de recurso
- ≋ Waterfall de requisições
- 🎬 Filmstrip — carregamento visual
- 🎯 Elemento LCP detalhado
- ⚡ Elementos CLS
- 💡 Oportunidades por categoria
- 🕐 Histórico salvo por navegador (localStorage)
- 🌙 Tema claro/escuro

## Deploy na Vercel

Conecte o repositório GitHub na Vercel e faça deploy automático.
